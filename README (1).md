## ✨ Features

- **Custom cursor** — cherry-red dot replacing the default pointer
- **Scrolling ticker** — animated skills strip
- **Hero section** — split layout with name, title, and stat panels on a Cherry Red block
- **Fade-in animations** — scroll-triggered reveal on every section
- **Fully responsive** — mobile-friendly layout
- **Single HTML file** — zero dependencies, no frameworks
